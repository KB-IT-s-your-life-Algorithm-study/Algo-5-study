## ✅ How to commit
- 스터디 시작 전 커밋, 풀지 못했을 땐 스터디 종료 후 커밋
- 멤버 모두 main 이용

- 파일 추가
  - sol : add P문제이름_이름 sol 
  - fail : add P문제이름_이름 fail
- 파일 수정
  - fix P문제이름_이름 
- 사이트 별 파일이름  
  - 프로그래머스 : P  
  - 백준 : B
